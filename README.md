# 🖼️ Steganography Tool in C with Docker

A powerful steganography tool built in C that hides and extracts secret data inside BMP images using the **LSB (Least Significant Bit) technique**, fully containerized using Docker for portability and ease of use.

---

## 🚀 Features

- 🔐 Encode secret files into BMP images  
- 🔍 Decode hidden data from stego images  
- 📁 File validation and error handling  
- ⚡ Efficient bit-level manipulation using LSB technique
